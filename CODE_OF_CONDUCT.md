# Contributor Covenant Code of Conduct

## 我们的承诺

为了营造一个开放、友好的社区,我们作为贡献者与维护者承诺:无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向,参与本项目都不会受到骚扰。

我们承诺以促成开放、欢迎、多元、包容、健康社区的方式行事和互动。

## 我们的标准

促进积极社区氛围的行为示例:

- 对他人表现出同理心与善意
- 尊重不同意见、观点与经验
- 给出并优雅地接受建设性反馈
- 为错误负责, 向受影响者道歉, 并从经验中学习
- 关注的不只是个人, 还有社区整体最佳利益

不可接受的行为示例:

- 使用带性暗示的语言或图像, 以及任何形式的性挑逗
- 钓鱼 (trolling)、侮辱或贬损评论、人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息 (如住址、电子邮件)
- 在专业场景中合理认定为不当的其它行为

## 维护者责任

社区维护者负责澄清并执行可接受行为的标准,并对其认为不当、具有威胁、冒犯或有害的任何行为采取适当且公平的纠正措施。

维护者有权移除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、Issue 及其它贡献, 并在适当时机说明审核决定。

## 适用范围

本行为准则适用于所有社区空间, 也适用于个人在公共空间正式代表社区时的行为。代表本社区的示例包括: 使用项目官方邮件地址、通过项目官方社交媒体账号发帖、或在线上/线下活动中担任指定代表。

## 执行

侮辱、骚扰或其它不可接受的行为可发送邮件到 **carlo.g.wang@outlook.sg** 报告社区领导以供执行。所有投诉将获得及时且公正的审查与调查。

所有社区领导都有义务尊重举报者的隐私与安全。

## 执行准则

社区领导将依据下列社区影响准则, 判断违反本行为准则的行为应受到的后果:

### 1. 更正

**社区影响**: 使用不当语言或在社区中被认为不专业或不受欢迎的其它行为。

**后果**: 由社区领导发出私下书面警告, 说明行为不当并解释为何不合适。可能要求公开道歉。

### 2. 警告

**社区影响**: 通过单次事件或一系列行为违反规范。

**后果**: 警告并附带后续行为的处置说明。在指定时间内不得与相关人员主动互动, 也包括在社区空间和外部渠道。违反该条件可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**: 严重违反社区准则, 包括持续的不当行为。

**后果**: 在指定时间内禁止与社区进行任何互动或公开通讯。在此期间, 不允许与相关人员进行任何公开或私下互动。违反该条件可能导致永久封禁。

### 4. 永久封禁

**社区影响**: 表现出违反社区准则的固定模式, 包括持续的不当行为、骚扰特定个人、或对特定群体的攻击与贬损。

**后果**: 永久禁止在社区中进行任何形式的公开互动。
